CodeSV

Sobre o Projeto

O CodeSV é uma plataforma moderna desenvolvida para simplificar a forma como programadores armazenam, partilham e exportam os seus snippets de código favoritos.

Exportação Estética

Gere imagens PNG de alta resolução com estilo "Dark/Hacker" para partilhar nas redes sociais.

Login Social

Acesso rápido e seguro através da sua conta GitHub, sem necessidade de novos registros.

Editor Poderoso

Experiência de edição fluida com o Monaco Editor e Syntax Highlighting via Prism.js.

Tecnologias Utilizadas
Java 21 Spring Boot 3 PostgreSQL Docker Thymeleaf Bootstrap 5
